API¶

Overview¶

The primary and secondary analysis pipelines must define and register their outputs via a common api. The cellhub.tasks.api module provides the code for doing this. The api comprises of an “api” folder into which pipeline outputs are symlinked (by the “register_dataset” “api” class method).

The api provides a standardised interface from which the downstream pipelines can access the information.
